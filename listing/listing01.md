###Что выведет программа? Объяснить вывод программы.

```go
package main
 
import (
    "fmt"
)
 
func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

###Ответ:



a[start:end] - такая запись возвращает срез массива с элемента 
start включительно, до элемента end => [77 78 79]
